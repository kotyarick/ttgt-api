# TTGT API

### Настройка

```
python -m venv .venv
.venv\bin\activate.ps1
pip install -U pip
pip install -r requirements.txt
fastapi run main.py
```

### Запуск для разработки

Чтобы сервер перезапускался при изменениях
`fastapi dev main.py`

### Инструменты системного администратора

`python manager.py`
